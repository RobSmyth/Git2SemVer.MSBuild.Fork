#  Prerequisites

Git2SemVer requires:

* `git` CLI to be executable from any project directory.
* `dotnet` CLI to be executable from any project directory.

Known compatibility:

* `dotnet.exe` `8.0.403` or later.
* `git` `2.41.0` or later.
* Windows 11
* Ubuntu 20.04 LTS
